# JPA를 이용한 게시판

# 프로젝트 설명
Spring Boot, JPA 를 공부하기 위해 만든 프로젝트 입니다.
기본적인 게시판을 둘러볼수 있습니다.

## 개발환경
+ IntelliJ IDEA
+ Java 11
+ Spring Boot 2.7.6

## 기술 세부 스택
+ Spring Web
+ Spring Data JPA
+ Thymeleaf
+ H2 Database
+ Lombok
+ Spring Security
+ BootStrap


## 기능
+ 회원 가입
+ 회원 수정
+ 회원 탈퇴
+ 게시글 작성
+ 게시글 수정
+ 게시글 삭제
+ 댓글 작성
+ 댓글 수정
+ 댓글 삭제
